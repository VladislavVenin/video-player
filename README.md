# Видео плеер
Собранный по уроку плеер

Посмотреть на плеер без скачивания и запуска сервера можно по [этой ссылке](https://vladislavvenin.github.io/video-player/)
## Требования
Для запуска сайта с плеером требуется:

- python3
- livereload 2.7.1
```
pip install livereload==2.7.1
```
## Запуск
Для запуска сайта с плеером используйте команду:
```
python server.py
```
далее перейдите по ссылке [`http://127.0.0.1:5500/`](http://127.0.0.1:5500/)
